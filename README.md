# TC cliente 30/03/2026

Ejercicio fetch con jsonplaceholder.

Escribe un programa que dado un id de usuario obtenga el nombre del usuario, los títulos de sus álbumes y los títulos de las fotos de cada album.

La información debe recuperarse en el menor tiempo posible, de forma paralela cuando sea posible.

Todos los *fetch* deben de controlar errores y asegurarse de que la respuesta es correcta.
